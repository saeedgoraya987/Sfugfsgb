# Ximagine Pro - AI Video Generation Engine

Deployable on Vercel - Generate AI videos from text prompts for free!

## Features

- 🎬 Text-to-Video Generation
- 🖼️ Image-to-Video Conversion
- 🎨 Multiple Video Styles (Standard, Fun, Spicy)
- 📐 Multiple Aspect Ratios (1:1, 16:9, 9:16)
- 🌐 Serverless API on Vercel
- 📱 Responsive UI
